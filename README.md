# Mi Portfolio

Pequeña API desarrollada en Spring BOOT con MySQL y JPA

## Referencias

### Study 👀

#### Traer todos estudios y logros academicos

```http
  GET /study
```

#### Traer un solo elemento

```http
  GET /study/{id}
```
